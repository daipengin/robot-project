実験3(PWMによるモーター制御)
内容

    3,次に低い音を鳴らす。それから5秒間低速で左右のモーターを回す
    4,最後、音を出して停止する
    ※制約
        モーターの動作にはrun(1000)のみを使い、数値は変更しないこと
